## Project Name: Country Management System (IT202)
### Project Summary:
A web application that allows users to register, log in, and manage their associated countries. Admins can assign roles, manage users, and add country data manually or  the CountryWise API. The app features role based access, modern UI, and integration with an external API for up to date country information.
